# Zeta

A Kotlin build system


## Installation

```bash
sudo cd /tmp
sudo git https://github.com/bit-sync/zeta.git 
sudo cd zeta
sudo rm /usr/bin/zeta
sudo mv zeta .zeta /usr/bin
```

## Create A Project

```bash
zeta init
```

## Build A Project

```bash
zeta run
```

## Build A native binary (Certain requirements don't use for now)

```bash
zeta native
```
